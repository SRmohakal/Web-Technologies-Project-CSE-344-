# Learning Management System (LMS)

## Introduction
The **Learning Management System (LMS)** is a web-based application designed to manage online courses, students, and teachers efficiently.  
It allows:
- **Admins** to manage courses and users.  
- **Teachers** to upload study materials and assignments.  
- **Students** to enroll in courses, view content, and track their progress.

---

## Objectives
- Provide an online platform for teaching and learning.  
- Allow course creation, content upload, and student enrollment.  
- Manage assignments, quizzes, and results.  
- Offer subscription for premium courses.

---

## Features
- **User Authentication:** Admin, Teacher, and Student login.  
- **Course Management:** Create, update, and delete courses.  
- **Content Upload:** Upload videos, PDFs, and notes.  
- **Student Management:** Enrollment and progress tracking.  
- **Cloud Storage:** Media stored using Cloudinary.

---

## Technology Stack
- **Frontend:** React.js, Tailwind CSS, DaisyUI  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Media:** Cloudinary  

---

## Expected Outcomes
- A fully functional web-based LMS platform.  
- Students can enroll, learn, and track their progress.  
- Teachers can create and manage courses efficiently.  
- Admin can control and manage the entire system.
